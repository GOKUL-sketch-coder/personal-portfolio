# personal-portfolio
🚀 Features
💻 Built with Next.js & Tailwind CSS – Modern frameworks for speed and design flexibility
📱 Responsive Design – Works seamlessly on desktop, tablet, and mobile
⚡ High Performance – Fast page loads with optimized routing and server-side rendering
🛠 Reusable Components – Clean, maintainable, and modular code
✨ Smooth UI/UX – Interactive navigation and polished design elements
📂 Project Structure
/personal-portfolio
├─ /components   # Reusable React components
├─ /pages        # Next.js pages
├─ /public       # Images, icons, and static assets
├─ /styles       # Tailwind CSS configurations
└─ README.md
🔧 Tech Stack
Framework: Next.js
Styling: Tailwind CSS
Deployment: Vercel / GitHub Pages
Version Control: Git & GitHub
📌 Getting Started
Clone the repository
git clone https://github.com/GOKUL-sketch-coder/personal-portfolio.git
cd personal-portfolio
Install dependencies
npm install
Run locally
npm run dev
Open http://localhost:3000
 to view it in the browser.
